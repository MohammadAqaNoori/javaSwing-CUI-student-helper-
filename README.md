# javaSwing-CUI-student-helper
## A fully functional Java Swing application with OOP, file handling, and professional GUI.

## Features :
Full OOP: Student class, FileHandler class, inheritance, encapsulation.

CRUD with file handling: Save, search, update, delete.

GPA calculation + classification with COMSATS rules.

Swing GUI: modern layout, scrollable output, popups.

Validation for input (marks, subjects, name).
